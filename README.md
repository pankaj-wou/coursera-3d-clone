# LearnVerse 3D — Interactive Learning Platform

A 3D interactive learning platform combining a Coursera-like web portal with immersive Babylon.js 3D environments featuring an AI avatar tutor.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
